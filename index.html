Where is the International Space Station right now?









Note: This API serves content over an HTTP connection which is insecure. Since Chrome and other browsers block insecure connections, I had to manually allow Chrome to serve mixed content to display the API information.

I was searching through various APIs and found one that could give me the current location of the International Space Station. I love all things outer space-related so I decided to use data from there: http://open-notify.org/Open-Notify-API/ISS-Location-Now/. Once I was able to get the coordinates, I thought it would be even more interesting to include a visual element and display the lat/long location on a map. I found out that Mapquest would allow me to do this, so I signed up for a free developer account to get an API key. I used Mapquestâ€™s Static Map API to render a map with a pin at given coordinates: https://developer.mapquest.com/documentation/static-map-api/v5/map/#basic-parameters. The map API is especially helpful in providing a visual representation of the ISS data collected. Having real-time access to this information could be helpful to scientists or researchers who are tracking the movements of the ISS. It could also be useful to people curious about outer space who want to know when the ISS is passing over them so they can search for it in the sky--there are applications that display live information about planets and other objects in the solar system, so this API could be helpful in that case as well.
